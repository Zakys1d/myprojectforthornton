How to Embed a Service-Hosted Video on a Website

This tutorial shows how to embed a video (example: YouTube) using the embed code. Each step includes a screenshot.

Before you start

  • Choose a video that allows embedding (no mature content).
  • Create an images folder for your screenshots.
  • Crop each screenshot to show only the important area.

Step 1: Open the video page

Go to the video hosting site (like YouTube) and open the video you want to embed.

Step 2: Click Share

Under the video, click the Share button. This opens sharing options.

Step 3: Choose Embed

In the share options, select Embed. If you do not see Embed, the uploader may have disabled embedding, so pick a different video.

Step 4: Copy the embed code

Copy the iframe embed code. It usually starts with <iframe.

Step 5: Paste the code into your HTML

Open your HTML file in your code editor and paste the embed code where you want the video to appear (usually inside <main>).
